Projet : Cookie Defense

[Map de base pour le projet:](./project-td-clicker/public/assets/map1.png)

Description :

- Tower defense basé sur un clicker pour progresser dans les niveaux en achetant des tours qui tirent sur des cookies. Ces mêmes cookies essayent d'arrivée au bout du parcours afin provoquer la fin de la partie. L'objectif du joueur est de survivre le plus longtemps possible contre des vagues d'ennemies infini de plus en plus forte afin de maximiser son score.

Fonctionnalité principales :

- Tuile de placement de tour
- Tour qui tire des projectiles
- Projectile téléguidé vers les cookies présents dans la range de la tour
- Round Infini de cookie de plus en plus difficile
- Score de survie stocké dans le local storage
- Cookies qui suivent un chemin tracé
- Main Screen
- Game Over Screen
- Écran de jeu fonctionnel
- Système de clicker avec argent pour acheter des tours
- Map originale
